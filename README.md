# Лабораторная работа №3: Два указателя для поиска пары с заданной суммой
**Вариант №30**

## Описание проекта
Приложение разработано на платформе Windows Forms (C#) и предназначено для комплексного математического анализа чисел. Программа позволяет классифицировать числа по четырем критериям, проводить статистический анализ диапазонов и визуализировать результаты.

### Основной функционал:
1. **Индивидуальный анализ**: Проверка одного числа на все заданные свойства.
2. **Анализ диапазона**: Массовая проверка чисел в заданном интервале.
3. **Фильтрация результатов**: Возможность просмотра только определенных типов чисел (например, только совершенных).
4. **Визуализация**: Построение динамического графика распределения свойств с поддержкой прокрутки (Scrolling).
5. **Тестирование**: Встроенный модуль автоматизированного тестирования алгоритмов.

---

## Математический аппарат (Вариант 30)
Программа реализует проверку следующих типов чисел:
*   **Простые числа**: Делятся только на 1 и на самих себя.
*   **Совершенные числа**: Равны сумме всех своих собственных делителей (6, 28, 496...).
*   **Палиндромы**: Читаются одинаково в обоих направлениях (121, 555...).
*   **Числа Армстронга**: Сумма цифр, возведенных в степень количества цифр, равна самому числу (153, 370...).
